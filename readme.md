 # Branch

**master** : 마지막에 dev와 vm-ui 를 merge하는 브랜치

**dev** : PR 보내는 브랜치

**feature/ui** : HTML, CSS 작성 + 상품 데이터 렌더링

**feature/product-view** : 상품 화면 작동 기능

**feature/product-select** : 상품 선택 화면 작동 기능

**feature/wallet** : 지갑 화면 작동 기능

**feature/contoller** : 금액 데이터 제어 

# 코드 짜는 규칙

- 클래스 위주로 작성하기
- 작은 단위의 기능은 함수로 작성하기
